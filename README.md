## 缠花铺线器

1、四股线永远平行
2、无需再理线和接线

考虑到成本问题，最终选用卡纸制作。
当然也可以做成3D打印方案

请用不干胶打印图纸后粘贴到铜版卡纸上组装
线卷内装的是标准家用机梭心

#### 材料：

​	不干胶打印的图纸
​	350g铜版卡纸
​	透明胶
​	橡皮筋
​	5mm小吸管

#### 工具：

​	剪刀、刻刀、迷你手钻
